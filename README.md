# Predicting LaLiga match outcomes

Predicting if a team will win using a random forest classifier on the (2019-2025) dataset from Kaggle

## The link to the dataset:

[Kaggle](https://www.kaggle.com/datasets/marcelbiezunski/laliga-matches-dataset-2019-2025-fbref)

## Model

- **Algorithm:** Random Forest
- Achieved **65% precision** on predicted wins
